# test-Vagrantfile

Vagrantfile for launch test ubuntu box

## Requirements
 - virtualbox (other providers is not tested)
 - vagrant
 - ansible

## Usage

```sh
git clone https://github.com/burykinne/test-Vagrantfile.git
cd test-Vagrantfile
vagrant up
```
Open http://127.0.0.1:3000 in your favourite browser
